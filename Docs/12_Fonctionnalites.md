# Objectif

Ce document décrit les cas de tests fonctionnels du moteur de calcul.

Chaque évolution du moteur devra être validée à l'aide de ces scénarios.

Les résultats attendus constituent la référence officielle.

---

# Structure d'un cas de test

Chaque cas comporte :

- Identifiant
- Description
- Profil de règles
- Nombre de joueurs
- Contrat
- Résultat de la donne
- Bonus
- Malus
- Résultat attendu
